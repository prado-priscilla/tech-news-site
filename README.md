# Tech News

A simple website built with HTML and CSS focused on sharing news and trends about technology, artificial intelligence, and games.

# Live Demo

🔗 [https://prado-priscilla.github.io/tech-news-site/](https://prado-priscilla.github.io/tech-news-site/)

# Features

* Home page with site introduction
* News page with technology-related content
* Contact page with links to social media and projects
* Navigation between pages
* Clean and modern layout

# Technologies Used

* HTML5
* CSS3

# 📁 Project Structure

* `index.html` → Home page
* `noticias.html` → News page
* `contato.html` → Contact page
* `style.css` → Styling
* `favicon/` → Website icon

# Purpose

This project was created for practicing HTML and CSS, focusing on layout organization and web page structure.
